About Album Browser

Album Browser is a single-page application built with Angular that allows you to explore albums and photos using the JSONPlaceholder API.

Features

  • Multi-page SPA with Angular Router
  • HTTP requests to a REST API
  • CRUD operations (Read, Update, Delete)
  • Reactive data handling with Observables
  • Service layer for API communication
  • Responsive design with scoped styles
